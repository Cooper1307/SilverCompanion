# 银龄小助手 (SilverCompanion) 电脑配置教程 v1.0

各位同学，我是陈露。为了保证项目顺利落地，请大家尽快完成以下“基建工程”：

## 1. 网络环境 (网络正义)

* **工具**：快连 (LetsVPN) - 推荐由组长提供的渠道。
* **要求**：确保能够顺畅访问 `github.com` 和 `openai.com` 等网站，这关系到我们调用阿里云百炼 API 的稳定性。

## 2. 编程利器 (IDE)

* **软件**：[Visual Studio Code](https://code.visualstudio.com/)
* **必装插件 (Extensions)**：
  * `Chinese (Simplified)` (中文包)
  * `Python` (后端开发必备)
  * `Volar` (Vue 3 官方支持)
  * `Tailwind CSS IntelliSense` (适老化 UI 快速开发)

## 3. 开发语言环境

* **Python**: 安装 [Python 3.10+](https://www.python.org/)。注意：安装时勾选 **"Add Python to PATH"**！
* **Node.js**: 安装 [Node.js v18+](https://nodejs.org/) (选 LTS 版)，用于驱动 Vue 前端。

## 4. AI 辅助神器

* **Antigravity/Cursor**: 这是我们提升代码产出效率的核心工具。
* **操作**：像使用浏览器一样简单，登录后可以在对话框里直接描述需求生成代码。
